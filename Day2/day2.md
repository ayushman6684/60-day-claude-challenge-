# Day 2: Prompt Engineering — Lazy vs Engineered Prompt

## What I Learned
Prompt Engineering is the skill of writing clear, specific instructions to get better responses from AI tools like Claude.

## Lazy Prompt
"Create an image explaining Prompt Engineering"

**Output:** Generic, unstructured explanation with no depth.

## Engineered Prompt
"You are an AI educator teaching complete beginners. Explain Prompt Engineering in simple language. Include: what it is, why it matters, weak vs strong prompt example, 3 benefits. Create a LinkedIn-ready 1080x1080 image with Claude brown/beige colors."

**Output:** Structured explanation with sections, examples, and a detailed image generation prompt.

## Key Difference
The engineered prompt gave role + context + format + requirements.
The lazy prompt gave none of that.

## What Surprised Me Most
How much structure changes the output quality — same AI, completely different results just from better instructions.
